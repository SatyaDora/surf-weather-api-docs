## Troubleshooting

### Problem: Dashboard not loading

**Possible causes**

- Slow internet connection
- Temporary server delay

**Solution**

1. Refresh the page.
2. Clear browser cache.
3. Try again after a few minutes.

---

### Problem: No weather data displayed

**Possible causes**

- Invalid location
- Data provider delay

**Solution**

1. Select another nearby location.
2. Check again after refresh.

---

### Problem: Alerts not working

**Possible causes**

- Alerts disabled in settings
- Notification permissions blocked

**Solution**

1. Go to Settings → Alerts.
2. Enable notifications.
3. Allow browser notifications.
