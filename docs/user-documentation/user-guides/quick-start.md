# Quick Start

Welcome to Surf Weather.

This guide helps you start using the platform in under 5 minutes.

## Step 1 — Sign In

Log in using your registered email and password.

## Step 2 — Select Location

Use the search bar to find your coastal city or beach.

## Step 3 — View Surf Conditions

The dashboard displays:

- Wave height
- Wind speed
- Temperature
- Tide levels

## Step 4 — Save Favorite Locations

Click ⭐ to save locations for quick access later.
