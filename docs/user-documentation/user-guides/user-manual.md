# User Manual

This manual explains all major features available to users.

## Dashboard Overview

The dashboard provides real-time surf and weather insights.

### Key Sections

- Current Conditions
- Hourly Forecast
- Weekly Forecast
- Alerts Panel

## Location Search

Use the search bar to switch between locations.

## Alerts

Users can enable alerts for:

- High waves
- Strong winds
- Storm warnings

## Account Settings

From the profile menu you can:

- Change units (Celsius/Fahrenheit)
- Manage saved locations
- Update profile details
