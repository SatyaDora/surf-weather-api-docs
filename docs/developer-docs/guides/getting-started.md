# Getting Started

Welcome to the Surf Weather API.

This guide helps you make your first successful API request.

---

## Prerequisites

Before you begin:

- Create an OpenWeather account
- Generate an API key
- Install a REST client (Postman or curl)

---

## Base URL

`https://api.openweathermap.org/data/2.5`


---

## Best Practices

- Never expose API keys publicly
- Store keys in environment variables
- Rotate keys periodically

---

## Common Errors

| Code | Meaning |
|---|---|
| 401 | Invalid API key |
